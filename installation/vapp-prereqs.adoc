= Virtual infrastructure requirements
:hardbreaks:
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ../media/

[.lead]
There are a few virtual infrastructure configurations that are required for installing Keystone Collector on your VMware vSphere systems.

.Prerequisites for the Keystone Collector server VM:

* Operating system: VMware vCentre server and ESXi 6.5 or later
* Core: 4 CPUs
* RAM: 12 GB RAM
* Disk space: 200 GB vDisk
